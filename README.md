### Public workbench

| Surface | What it is | Status |
| --- | --- | --- |
| [`lazy-skill-router`](https://github.com/chowonje/lazy-skill-router) | Deterministic Codex skill recommendation layer that keeps authorization outside the router. | v0.5 dev |
| [`knowledge-hub`](https://github.com/chowonje/knowledge-hub) | Local-first research runtime for papers, web sources, and project context with reviewable evidence. | research preview |
| [`paper-cards-skill`](https://github.com/chowonje/paper-cards-skill) | Local PDF to Korean or English Markdown research cards with a repeatable QA path. | preview |
| [`project-rule-skill-share`](https://github.com/chowonje/project-rule-skill-share) | Codex skill for bootstrapping governance files, GitHub templates, ADRs, and eval scaffolds. | public skill |

### How I build

- Keep data and control local when possible.
- Treat generated output as a review candidate, not verified truth.
- Prefer small, reversible changes with visible evidence.
- Separate recommendations, permissions, and execution.
