### [Knowledge Hub](https://github.com/chowonje/knowledge-hub)

A local-first, evidence-first research knowledge runtime for papers, web sources, and project context.

A Codex skill for bootstrapping project governance, contributor guidance, GitHub templates, ADRs, and evaluation scaffolds.

## What I care about

- Evidence and verification instead of opaque automation

- Clear boundaries between recommendations, permissions, and execution
