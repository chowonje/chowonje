## About

I build small, inspectable tools for AI-assisted research and software development. My work focuses on local-first systems, evidence trails, and explicit boundaries between recommendations, permissions, and execution.

| Project | What it does |

## Working principles

- Keep data and control local when possible.
- Treat generated output as a review candidate, not as verified truth.
- Prefer small, reversible changes with visible evidence.
- Separate recommendations, permissions, and execution.
